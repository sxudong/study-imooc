#### 题目要求
有如下文本，其中前5行内容为
```
1111111:13443253456
2222222:13211222122
1111111:13643543544
3333333:12341243123
2222222:12123123123
```
用shell脚本处理后，按下面格式输出：
```
[1111111]
13443253456
13643543544
[2222222]
13211222122
12123123123
[3333333]
12341243123
```

#### 参考答案
```
#!/bin/bash
#这个脚本用来处理文本
#作者：猿课-阿铭 www.apelearn.com
#日期：2018-12-12

for w in `awk -F ':' '{print $1}' 3.txt |sort |uniq`
do 
    echo "[$w]"
    awk -v w2=$w -F ':' '$1==w2 {print $2}' 3.txt
done
```
