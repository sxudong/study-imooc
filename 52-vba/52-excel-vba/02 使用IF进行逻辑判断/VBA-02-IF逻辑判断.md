# 02 第二课 使用IF进行逻辑判断
## 1）IF条件判断式语法

```vb
IF……Then // if 如果怎么样, then就怎么样
   ……
Else            // 否则
   ……
End IF        // 结束if

IF……Then
   ……
ElseIF        // 否则 if 如果
```
